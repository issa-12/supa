# Highlighted Requirements

## Major Requirements (6)

1. **Advanced analytics dashboard with data visualization**
   - Interactive charts and graphs (line, bar, pie, etc.).
   - Real-time data updates.
   - Export functionality (PDF, CSV, etc.).
   - Customizable date ranges and filters.

2. **Recommendation system using machine learning**
   - Personalized recommendations based on user behavior.
   - Collaborative filtering or content-based filtering.
   - Continuously improve recommendations over time.

3. **Standard user management and authentication**
   - Users can update their profile information.
   - Users can upload an avatar (with a default avatar if none provided).
   - Users can add other users as friends and see their online status.
   - Users have a profile page displaying their information.

4. **Use a framework for both the frontend and backend**
   - Use a frontend framework (React, Vue, Angular, Svelte, etc.).
   - Use a backend framework (Express, NestJS, Django, Flask, Ruby on Rails, etc.).

5. **Implement real-time features using WebSockets or similar technology**
   - Real-time updates across clients.
   - Handle connection/disconnection gracefully.
   - Efficient message broadcasting.

6. **A public API to interact with the database with a secured API key, rate limiting, documentation, and at least 5 endpoints**
   - `GET /api/{something}`
   - `POST /api/{something}`
   - `PUT /api/{something}`
   - `DELETE /api/{something}`

## Minor Requirements (7)

1. **Content moderation AI** (auto moderation, auto deletion, auto warning, etc.).

2. **Sentiment analysis for user-generated content.**

3. **User activity analytics and insights dashboard.**

4. **Implement remote authentication with OAuth 2.0** (Google, GitHub, 42, etc.).

5. **Implement advanced search functionality with filters, sorting, and pagination.**

6. **Support for multiple languages (at least 3 languages)**
   - Implement i18n (internationalization) system.
   - At least 3 complete language translations.
   - Language switcher in the UI.
   - All user-facing text must be translatable.

7. **Right-to-left (RTL) language support**
   - Support for at least one RTL language (Arabic, Hebrew, etc.).
   - Complete layout mirroring (not just text direction).
   - RTL-specific UI adjustments where needed.
   - Seamless switching between LTR and RTL.

## Totals

- **Major:** 6
- **Minor:** 7
- **Total:** 13
